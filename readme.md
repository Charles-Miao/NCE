# 此项目的由来

## 2010s
上学时英语学习比较一般，正常也就70多分。工作后一直有一个将英语学好的想法，希望能够达到正常交流的水平。2010s有透过墨墨背单词来学习NCE的单词，以及VOA的文章来学习英语的听读，但是终因工作和生活，并未坚持下来。
